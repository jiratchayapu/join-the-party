# Getting Started with Join the Party Web Application

## Available Scripts

In the project directory, you can run:

### Frontend

`$ cd ../frontend` 

#### install dependencies
`$ yarn install` 

#### Runs the app in the development mode.
`$ yarn start` 

open http://localhost:3000 to view it in your browser.

### backend

`$ cd ../backend` 

#### install dependencies
`$ yarn install`

#### run server
`$ yarn start`
